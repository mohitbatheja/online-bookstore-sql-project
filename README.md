# 📚 Online Bookstore SQL Project

## 📌 Overview

The **Online Bookstore SQL Project** is a relational database project developed using **MySQL** to demonstrate core SQL concepts through real-world business scenarios. The project focuses on designing a bookstore database, performing data manipulation, and solving analytical business queries.

It showcases practical SQL skills required for **Data Analyst**, **Business Analyst**, and **SQL Developer** roles.

---

# 🎯 Objectives

- Design a relational database for an online bookstore.
- Perform CRUD operations using SQL.
- Practice DDL, DML, and DQL statements.
- Analyze bookstore sales and inventory using SQL queries.
- Strengthen SQL problem-solving skills through business scenarios.

---

# 🗂️ Database Schema

The project consists of three tables:

| Table | Description |
|--------|-------------|
| **Books** | Stores book information such as title, author, category, price, stock, and publication year. |
| **Customers** | Stores customer details including name, city, email, and signup date. |
| **Orders** | Stores customer purchase details including quantity, order date, and total amount. |
